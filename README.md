# FK Icons

## Instalação

### npm

```console
$ npm install fk-icons --save
```

### Bower

```console
$ bower install fk-icons --save
```

## Adicionar mais ícones

1. Copie o arquivo SVG nas pasta SVG.
2. No console execute os comandos:

```console
$ npm install fk-icons --save
```

```console
$ gulp
```
